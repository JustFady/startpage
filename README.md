# Dashboard Startpage

**Live Demo:** [https://justfady.github.io/startpage/](https://justfady.github.io/startpage/)

A personal browser startpage with a glass dashboard layout, quick links, search, weather, and tasks. It is built as a static site and published with GitHub Pages.

## Features

- Responsive glass layout for desktop, half-width windows, and phones.
- Auto-focused Google search so typing on the page goes straight into search.
- Four click-to-cycle background themes.
- Auto-detected weather from wttr.in with cached last-known value.
- Lightweight CPU and memory status display.
- Persistent task list with checked/completed states and delete controls.
- Quick-link cards for general sites, dev tools, media, and tasks.
- Automatic compact spacing on smaller or shorter screens.
- Web app manifest and custom icons for adding the startpage to a phone home screen.

## Tech

- HTML, CSS, and vanilla JavaScript.
- GitHub Pages for hosting.
- wttr.in for weather.
- Inter and JetBrains Mono fonts.

## Local Use

Open `index.html` directly in a browser, or serve the folder with any static file server.
