# 🌌 Dashboard Startpage

A premium, high-performance browser startpage designed with **Glassmorphism** aesthetics and dynamic theme integration.

---

### 🚀 Key Features
- **🖼️ 4K Theme Cycle**: 4 distinct, high-quality aesthetic backgrounds (Cars, Space, Nature, Anime).
- **☁️ Real-time Weather**: Live weather updates for your location via the `wttr.in` API.
- **⏱️ System Status**: Dynamic simulation of CPU and Memory usage for a high-tech dashboard feel.
- **📝 Persistent Tasks**: A built-in todo list with LocalStorage persistence and checkbox completion.
- **🔍 Global Search**: Integrated search bar with smooth transitions and Google integration.
- **📱 Fully Responsive**: Optimized for all screen sizes with a clean, centered layout.

### 🎨 Themes
Click the **profile picture** to cycle through the available themes:
1. **Blue**: Porsche 911 GT3 RS Aesthetic
2. **Green**: Cosmic Deep Space Nebula
3. **Purple**: Misty Pink Mountain Sunset
4. **Red**: Ichigo Kurosaki (Bleach) Aesthetic

### 🛠️ Technology Stack
- **Frontend**: Vanilla HTML5, CSS3 (Glassmorphism), JavaScript (ES6+)
- **Storage**: `localStorage` for theme and task persistence
- **API**: `wttr.in` for live weather data
- **Typography**: Inter, JetBrains Mono, and SpaceMono Nerd Font
