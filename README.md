# Startpage

A premium, specialized startup page with glassmorphism design, dynamic dashboard, and theme switching.

## Features
- **Modern Dashboard**: Real-time clock, date, and system status.
- **Theme Switching**: Click the profile photo to cycle through 4 premium themes (Classic Dark, Cyberpunk, Sunset, Forest).
- **Todo List**: Persistent task list saved locally.
- **Search**: Integrated search bar for quick access.

## Usage
Open `fadyy/index.html` in your browser or run a local server:
```bash
python3 -m http.server 8000
```
Then visit `http://localhost:8000/fadyy/`.
